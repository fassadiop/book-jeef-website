# BOOK JEEF Website

Site web vitrine + système d'adhésion pour la coopérative BOOK JEEF.

## 🔧 Stack
- HTML / Tailwind CSS
- JavaScript (vanilla)
- PHP (API)
- MySQL

## 🚀 Fonctionnalités
- Formulaire d’adhésion
- Formulaire de contact
- Backend PHP (API REST simple)
- Interface responsive premium

## 📁 Structure
- `/public` : pages frontend
- `/api` : backend PHP
- `/assets` : CSS / JS / images

## ⚙️ Installation
1. Placer le projet dans `htdocs`
2. Démarrer XAMPP (Apache + MySQL)
3. Accéder :
http://localhost/site-book-jeef/public/index.php
